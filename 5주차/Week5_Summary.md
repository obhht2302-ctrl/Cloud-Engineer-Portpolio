## 📌 IAM 핵심 개념 정리
- **User:** 실제 사람이 사용하는 계정  
- **Group:** 여러 User가 공유하는 권한 묶음  
- **Role:** AWS 서비스(EC2, Lambda)가 사용하는 자격 역할  
- **Policy:** JSON 문서 기반 권한 정의 방식  
- **MFA:** 계정 보호 필수 요소  
- **CloudTrail:** 모든 IAM 이벤트 기록  
- **Least Privilege:** 최소 권한 원칙  
- **STS:** 임시 보안 자격 증명  
- **Terraform:** IAM 자동화 도구
