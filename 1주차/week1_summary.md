# 🗓️ Week 1 – 학습 요약 보고서

## 📘 Day 1 – EC2, Git, Linux 기초

### 🌩️ AWS EC2 생성 & SSH 접속
- **목표:** EC2 인스턴스 생성 및 SSH 원격 접속 이해.
- **핵심 명령어:**
  ```bash
  ssh -i ~/mykey.pem ubuntu@<EC2_IP>
  ```
- **성과:** AWS 환경에서 첫 서버 구축 및 보안 접속 성공.

### 🧰 Git 설치 & 저장소 만들기
- **명령어:** `git init`, `git add .`, `git commit`, `git push`
- **성과:** GitHub 원격 저장소 연동 및 버전 관리 개념 이해.

### 🐧 리눅스 파일 & 디렉토리 명령어
- **명령어:** `pwd`, `ls -l`, `mkdir`, `touch`, `rm`
- **성과:** 파일 시스템 구조와 경로 개념 숙지.

---

## ⚙️ Day 2 – 사용자, 서비스, 프로세스, nginx

### 👤 사용자 & 권한 관리
- **명령어:** `adduser`, `chmod`, `chown`, `usermod -aG`
- **성과:** 사용자 생성 및 권한 제어 실습 완료.

### 🔧 서비스 관리
- **명령어:** `systemctl start|stop|status`, `journalctl -xe`
- **성과:** nginx 서비스 제어 및 로그 분석 수행.

### 🧠 프로세스 관리
- **명령어:** `ps aux`, `top`, `kill`, `pgrep`
- **성과:** 프로세스 모니터링 및 종료 실습.

### 🌐 nginx 외부 접속 확인
- **명령어:** `ss -tuln`, `curl -I`, AWS 보안 그룹 설정.
- **성과:** EC2 퍼블릭 IP를 통한 nginx 웹서버 접속 성공.

---

## 🌐 Day 3 – 네트워크와 패키지 관리

### 🧩 패키지 설치 & 업데이트
- **명령어:** `apt update`, `apt install`, `apt upgrade`
- **성과:** 시스템 최신화 및 네트워크 진단 도구 설치.

### 🌍 네트워크 명령어 실습
- **명령어:** `ping`, `curl`, `wget`, `netstat`, `ss`
- **성과:** 네트워크 연결 및 서비스 응답 확인.

### 🖥️ 네트워크 구조 시각화 (OSI 7계층)
- **성과:** 계층별 데이터 흐름(HTTP 요청 → IP 전송) 시각적 이해.

---

## 🧠 Day 4 – 자동화와 DNS

### 💻 Shell 스크립트 작성하기
- **명령어:** `chmod +x log.sh`, `./log.sh`
- **성과:** 서버 로그 자동 기록 스크립트 구현.

### ⏰ Crontab 사용하기
- **명령어:** `crontab -e`, `systemctl status cron`
- **성과:** 주기적 자동 실행(1분마다) 성공.

### 🌐 DNS 조회하기
- **명령어:** `nslookup`, `dig`, `dig +short`
- **성과:** DNS 질의 원리 및 IP 변환 구조 이해.

---

## 💪 Day 5 – 미니 프로젝트 & 복습

### 🧩 미니 프로젝트: 시스템 리포트 자동화
- **목표:** EC2 상태를 자동으로 점검하여 `/home/ubuntu/week1_project/report.txt`에 저장.
- **명령어:** `uptime`, `df -h`, `free -h`, `systemctl`, `ss`
- **성과:** 자동화 + Git 연동 완성.

### ⚙️ 실행 요령 및 복습 퀴즈
- **내용:** 보안 그룹, Git 충돌, cron, DNS 등 1주차 핵심 복습.
- **성과:** 실무형 서버 관리 능력 강화.
